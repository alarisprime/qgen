q·gen

q·gen generates files and folders from templates

View the Project on GitHub alarisprime/qgen

Every time when you want to add a new post for your Jekyll Blog, do you create file and copy paste the Front Matter? Or, you manually create files for the new React Component?

q·gen can generate new files or a folder of files from pre-defined templates.

Installation

$ npm install -g qgen
